# crazyoptimist's dotfiles

Everything is for Ubuntu/Debian by default, but you will be able to make it work for your distro with minimal adjustments.

### Shell Prompt(Starship)

```bash
make init
```

### Vim

```bash
make vim
```

### NeoVim

```bash
make nvim
```

```bash
pip install pynvim
npm i -g neovim
```
